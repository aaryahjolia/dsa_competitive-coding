## Code solution :

```cpp

#include<bits/stdc++.h>
using namespace std;
int main(){
    string a;
    cin>>a;
    // If first char uppercase, convert into lowercase !
    if(a[0]>96 && a[0]<123){
        a[0]=a[0]-32;
    }
    cout<<a<<endl;
return 0;
}
```
