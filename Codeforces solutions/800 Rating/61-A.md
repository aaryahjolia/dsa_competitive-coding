## Code solution :

```cpp

#include<bits/stdc++.h>
using namespace std;
int main(){
    // Input
    string a,b;
    cin>>a>>b;
    // Sasta(cheap) XOR algorithm
    for(int i=0;i<a.length();i++){
        if(a[i]==b[i])cout<<0;
        else cout<<1;
    }
return 0;
}

```
