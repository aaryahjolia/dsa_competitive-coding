## Question Link :

https://codeforces.com/problemset/problem/133/A

## Code solution :

```cpp

#include <bits/stdc++.h>
using namespace std;
int main(){
    // Taking input
    string s;
    cin >> s;
    // if any character is H,Q or 9, program will produce output, so checking it for characters and printing msg
    for (int i = 0; i < s.length(); i++){
        if (s[i] == 'H' || s[i] == 'Q' || s[i] == '9'){
            cout << "YES" << endl;
            return 0;
        }
    }
    cout << "NO" << endl;
    return 0;
}

```
