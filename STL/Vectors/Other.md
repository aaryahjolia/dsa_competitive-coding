## swap ()

```cpp
vector<int> v1 = {1,2,3,4,5};
vector<int> v2 = {6,7,8,9,10};
v1.swap(v2); // Swaps v1 and v2
```

### Output

```cpp
v1 = {6,7,8,9,10}
v2 = {1,2,3,4,5}
```

## clear ()

```cpp
vector<int> v = {1,2,3,4,5};
v.clear(); // Clears the vector
```

### Output

```cpp
v = {}
```

## empty ()

```cpp
vector<int> v = {1,2,3,4,5};
cout << v.empty(); // Returns 0 (false) if vector is not empty
```

### Output

```cpp
0
```
