## Description

Element can be removed from last using pop method, remember it will remove the last element only.

## Method

```cpp
stack<int> first;
// Adding Element for demo
first.push(1);
first.push(2);
first.push(4);
first.push(6);


// popping (removing) elements
first.pop(); // 6 removed
first.pop(); // 4 removed
```

### Stack now contains

```cpp
2
1
```
